### Run locally
1. Clone this repository:
```bash
git clone https://github.com/Surveinit/jp-resume-generator
```

2. Change directory
```bash
cd jp-resume-generator 
```

3. Install the required gems:
```bash
bundle install
```

4. Run the Rails server:
```bash
rails server
```
5. Visit the app in your browser:
```
http://localhost:3000
```
